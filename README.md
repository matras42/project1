# Опис стилів сторінки

Цей файл відповідає за зовнішній вигляд сторінки. Я створив чистий, професійний дизайн з зелено-земляною кольоровою гамою. Для оформлення використовуються два шрифти: Damion для логотипу (художній стиль) і Lato для всього тексту (легко читається). Вся сторінка має темно-зелений фон з кольором `#81a256`. 

Хедер і меню розташовані по краях сторінки, пункти меню підсвічуються при наведенні, а активний пункт має світліший фон. Секція "Our Work" — це велика презентація з білим заголовком і описом. Фільтри портфоліо вирівняні в один рядок із заголовком, при цьому активний фільтр ("Photography") виділений помаранчевим кольором. Галерея робіт розташована в сітці по 4 елементи в рядку, яка автоматично адаптується під розмір екрану. 

При наведенні на зображення з'являється зелений напівпрозорий фон з білим «плюсом» посередині, а при наведенні на підпис з'являється білий трикутник зверху. Всі ці елементи роблять сторінку зручною та зрозумілою для користувача.
